Expand description

A trait controlling how SharedMode and all associated types store reference-counted values.

Safety

When ptr_eq is provided two Containers from separate calls to new(), it always returns false. When ptr_eq is provided the same Container in both a and b, it always returns true.

Breaking this guarantee is undefined behavior.
